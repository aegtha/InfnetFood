Projeto desenvolvido para a disciplina de Desenvolvimento Mobile com React Native. O aplicativo consiste em uma plataforma de delivery completa, cobrindo desde a autenticação até o checkout.

## 🛠️ Exercícios Implementados (6 ao 15)

- **[Ex 6 e 11] Autenticação:** Fluxo de login com validação e separação entre rotas públicas e privadas.
- **[Ex 8 e 9] Mapas:** Integração com `react-native-maps` exibindo 10 pontos de interesse no Centro do RJ.
- **[Ex 10 e 15] API Externa:** Integração com a **API ViaCEP** para preenchimento automático de endereço via `fetch`.
- **[Ex 12 e 14] Feedback e Alertas:** Animações de sucesso e notificações locais (`expo-notifications`) para status do pedido.
- **[Ex 13] Tematização:** Implementação de Modo Claro e Modo Escuro via `Context API`.
